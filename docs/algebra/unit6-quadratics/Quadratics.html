6.2 🗂️ Formula Card

Quadratic Equations — Formula Card

Factorising (a = 1)

💡 Sum-Product Method Find two numbers that multiply to \(c\) and add to \(b\).

Standard form: \(x^2 + bx + c = (x + p)(x + q)\)

where \(p \times q = c\) and \(p + q = b\)

Examples: \[x^2 + 5x + 6 = (x+2)(x+3) \quad [2\times3=6,\;2+3=5]\] \[x^2 - x - 12 = (x+3)(x-4) \quad [3\times(-4)=-12,\;3+(-4)=-1]\]

Factorising (a ≠ 1) — ac Method

🎯 ac Method Multiply \(a \times c\), then find two numbers that multiply to \(ac\) and add to \(b\).

Steps: For \(ax^2 + bx + c\):

  1. Find \(p \times q = ac\) and \(p + q = b\)
  2. Split the middle term: \(ax^2 + px + qx + c\)
  3. Group and factorise

Example: \(2x^2 + 7x + 3\), \(\;ac = 6\), find \(p,q: p\times q=6,\; p+q=7 \Rightarrow 1,6\) \[2x^2 + x + 6x + 3 = x(2x+1) + 3(2x+1) = (x+3)(2x+1)\]

Special Forms

⚠️ Spot These First! Always check for special forms before using other methods.

Difference of Two Squares: \[a^2 - b^2 = (a+b)(a-b)\]

Perfect Square: \[a^2 + 2ab + b^2 = (a+b)^2 \qquad a^2 - 2ab + b^2 = (a-b)^2\]

Examples: \[x^2 - 25 = (x+5)(x-5)\] \[x^2 + 8x + 16 = (x+4)^2\]

Quadratic Formula

📝 Always works! Use when factorising is not possible.

For \(ax^2 + bx + c = 0\): \[x = \dfrac{-b \pm \sqrt{b^2 - 4ac}}{2a}\]

The Discriminant \(\Delta = b^2 - 4ac\):

\(\Delta > 0\) Two distinct real roots
\(\Delta = 0\) One repeated root
\(\Delta < 0\) No real roots

Example: \(x^2 - 5x + 3 = 0\) \[x = \dfrac{5 \pm \sqrt{25-12}}{2} = \dfrac{5 \pm \sqrt{13}}{2}\]

Completing the Square

Monic (\(a = 1\)): \(x^2 + bx + c = \left(x + \dfrac{b}{2}\right)^2 - \left(\dfrac{b}{2}\right)^2 + c\)

Non-monic (\(a \neq 1\)): \(ax^2 + bx + c = a\left(x + \dfrac{b}{2a}\right)^2 - \dfrac{b^2}{4a} + c\)

To solve: Write in form \((x+p)^2 = q\), then \(x = -p \pm \sqrt{q}\)

Example: \(x^2 + 6x + 2 = 0\) \[( x+3)^2 - 9 + 2 = 0 \implies (x+3)^2 = 7 \implies x = -3 \pm \sqrt{7}\]

\[\begin{align*} \text{Use index law: } (a^m)^n = a^{mn} \\ 2^{(x - 3)(x - 3)} = 2^{4} \\ \text{Equate exponents: } (x - 3)(x - 3) = 4 \\ x^2 -6x + 9 = 4 \\ x^2 -6x + 5 = 0 \\ x = \dfrac{-(-6) \pm \sqrt{(-6)^2 - 4(1)(5)}}{2} \\ x = \dfrac{6 \pm 4}{2} \\ x = 5 \text{ or } x = 1 \end{align*}\]
