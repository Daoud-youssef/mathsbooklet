Algebraic Indices & Surds

Laws of Indices — The Core Rules

💡 Pro Tip These rules ONLY apply when the base (the big letter) is the same.

Multiplication & Division

  1. Multiplication: Add the powers \(\rightarrow\) \[x^a \times x^b = x^{a+b}\]
  2. Division: Subtract the powers \(\rightarrow\) \[\frac{x^a}{x^b} = x^{a-b}\]

Examples: \[y^5 \times y^3 = y^{5+3} = y^8\] \[10w^7 \div 2w^3 = (10 \div 2)\,w^{7-3} = 5w^4\]

Power of a Power & Zero

🎯 Memory Trick Brackets mean multiply!
Anything to the power of \(0\) is a hero (it becomes \(1\)).

The Bracket Rule: \[(x^a)^b = x^{a \times b} = x^{ab}\]

Examples: \[(k^3)^4 = k^{3 \times 4} = k^{12}\] \[(5x^2)^3 = 5^3 \times x^{2 \times 3} = 125x^6\]

The Zero Rule: \[x^0 = 1\]

Negative & Fractional Indices

⚠️ Common Mistake A negative power does NOT make the number negative; it makes it a reciprocal (fraction)!

Negative Powers: \[x^{-a} = \frac{1}{x^a}\]

Fractional Powers: \[x^{\frac{1}{n}} = \sqrt[n]{x} \qquad x^{\frac{m}{n}} = \left(\sqrt[n]{x}\right)^m\]

Examples: \[4^{-2} = \frac{1}{4^2} = \frac{1}{16} \qquad 9^{\frac{1}{2}} = \sqrt{9} = 3 \qquad 8^{\frac{2}{3}} = \left(\sqrt[3]{8}\right)^2 = 2^2 = 4\]

Expanding Brackets with Indices

📝 Quick Method Multiply the “Big” numbers (coefficients), add the “Small” numbers (indices).

Single Brackets: \[a(b + c) = ab + ac\]

Examples: \[3x^2(2x^3 + 4) = (3 \times 2)x^{2+3} + (3 \times 4)x^2 = 6x^5 + 12x^2\] \[x^5(x^2 - x^{-3}) = x^{5+2} - x^{5-3} = x^7 - x^2\]

Algebraic Fractions with Indices

Simplifying Fractions:

  1. Simplify the numerator (top) first using multiplication laws.
  2. Simplify the denominator (bottom).
  3. Use the division law (subtract) to find the final result.

Example: \[\frac{a^5 \times a^4}{a^2} = \frac{a^{9}}{a^2} = a^{9-2} = a^7\]

Example with Coefficients: \[\frac{(2x^3)^2}{4x^4} = \frac{4x^6}{4x^4} = x^{6-4} = x^2\]

Surds — What is a Surd?

💡 Definition A surd is a root that cannot be written as an exact fraction — it is irrational.
\(\sqrt{2},\ \sqrt{3},\ \sqrt{5}\) are surds. \(\sqrt{4} = 2\) is not a surd.
Key fact: \((\sqrt{a})^2 = a\) always.

Operating with Surds

🎯 Key Idea You can only add or subtract surds with the same radicand. Think of \(\sqrt{3}\) as a unit — just like \(3x + 5x = 8x\), we get \(3\sqrt{3} + 5\sqrt{3} = 8\sqrt{3}\).

Addition & Subtraction — collect like surds only: \[a\sqrt{n} + b\sqrt{n} = (a+b)\sqrt{n} \qquad a\sqrt{n} - b\sqrt{n} = (a-b)\sqrt{n}\]

Unlike surds cannot be combined: \(\quad 3\sqrt{2} + 5\sqrt{3}\) stays as is.

Multiplication & Division: \[\sqrt{a} \times \sqrt{b} = \sqrt{ab} \qquad \frac{\sqrt{a}}{\sqrt{b}} = \sqrt{\frac{a}{b}}\] \[a\sqrt{m} \times b\sqrt{n} = ab\sqrt{mn}\]

Examples: \[5\sqrt{7} - 10\sqrt{7} + 2\sqrt{3} = -5\sqrt{7} + 2\sqrt{3}\] \[2\sqrt{10} \times 7\sqrt{6} = 14\sqrt{60} = 14\sqrt{4 \times 15} = 28\sqrt{15}\]

Simplifying Surds

⚠️ Always extract the largest perfect square factor.
\(\sqrt{72} = \sqrt{36 \times 2} = 6\sqrt{2}\) ✅ \(\quad\) not \(2\sqrt{18}\) ✗

Method — find largest perfect square \(k^2\) that divides \(n\): \[\sqrt{n} = \sqrt{k^2 \times r} = k\sqrt{r}\]

Steps: find square factor → split → evaluate \(\sqrt{k^2} = k\)

Examples: \[\sqrt{200} = \sqrt{100 \times 2} = 10\sqrt{2}\] \[3\sqrt{48} = 3 \times \sqrt{16 \times 3} = 3 \times 4\sqrt{3} = 12\sqrt{3}\]

Perfect Squares: \(4,\ 9,\ 16,\ 25,\ 36,\ 49,\ 64,\ 81,\ 100,\ 121,\ 144,\ 169,\ 196,\ 225\)

Expanding & Rationalising

📝 Expanding works exactly like algebra — distribute every term, use FOIL for double brackets, and remember \((\sqrt{a})^2 = a\).

Single Bracket: \[\sqrt{a}(\sqrt{b} + c) = \sqrt{ab} + c\sqrt{a}\]

Double Bracket (FOIL): \[(\sqrt{a} + b)(\sqrt{a} + c) = a + (b+c)\sqrt{a} + bc\]

Perfect Square: \[(\sqrt{a} + b)^2 = a + 2b\sqrt{a} + b^2\]

Conjugate Pair — always gives a rational result: \[(\sqrt{a} + \sqrt{b})(\sqrt{a} - \sqrt{b}) = a - b\]

Examples: \[\sqrt{3}(\sqrt{3}+2) = 3 + 2\sqrt{3}\] \[(\sqrt{5}+1)(\sqrt{5}-1) = 5-1 = 4\] \[(\sqrt{2}+3)^2 = 2 + 6\sqrt{2} + 9 = 11 + 6\sqrt{2}\]

Rationalising the Denominator — remove the surd from the bottom.

Simple: multiply top and bottom by \(\sqrt{b}\): \[\frac{a}{\sqrt{b}} \times \frac{\sqrt{b}}{\sqrt{b}} = \frac{a\sqrt{b}}{b}\]

Binomial: multiply by the conjugate of the denominator: \[\frac{a}{\sqrt{b}+c} \times \frac{\sqrt{b}-c}{\sqrt{b}-c} = \frac{a(\sqrt{b}-c)}{b-c^2}\]

The conjugate of \((\sqrt{b}+c)\) is \((\sqrt{b}-c)\) — flip the sign between the terms.

Examples: \[\frac{1}{\sqrt{3}} = \frac{\sqrt{3}}{3} \qquad \frac{6}{\sqrt{2}} = \frac{6\sqrt{2}}{2} = 3\sqrt{2}\]

\[\frac{5}{\sqrt{3}+2} \times \frac{\sqrt{3}-2}{\sqrt{3}-2} = \frac{5(\sqrt{3}-2)}{3-4} = \frac{5(\sqrt{3}-2)}{-1} = 10-5\sqrt{3}\]
