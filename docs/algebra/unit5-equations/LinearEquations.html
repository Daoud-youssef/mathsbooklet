5.2 🗂️ Formula Card

Linear Equations — Solving Strategies

The Balance Method

💡 Golden Rule Whatever you do to one side, you must do to the other side.

One-step \[x + 5 = 12 \;\Rightarrow\; x = 12 - 5 = 7\] \[3x = 21 \;\Rightarrow\; x = 21 \div 3 = 7\]

Two-step — undo addition/subtraction first, then multiplication/division \[2x + 3 = 11 \;\Rightarrow\; 2x = 8 \;\Rightarrow\; x = 4\]

Unknowns on Both Sides

🎯 Strategy Move all \(x\) terms to one side, all numbers to the other.

Method: \[5x + 2 = 2x + 8\] \[5x - 2x = 8 - 2\] \[3x = 6 \;\Rightarrow\; x = 2\]

Expanding Brackets

⚠️ Remember The number outside multiplies every term inside the bracket.

Single bracket: \[3(2x + 5) = 27 \;\Rightarrow\; 6x + 15 = 27 \;\Rightarrow\; x = 2\]

Two brackets: \[2(x + 3) + 4(x - 1) = 18\] \[2x + 6 + 4x - 4 = 18\] \[6x + 2 = 18 \;\Rightarrow\; x = \dfrac{8}{3}\]

Algebraic Fractions

📝 Key Move Multiply every term by the denominator (or LCM) to clear fractions first.

Single fraction: \[\dfrac{2x + 1}{3} = 5 \;\Rightarrow\; 2x + 1 = 15 \;\Rightarrow\; x = 7\]

Fractions on both sides — cross multiply: \[\dfrac{x + 1}{2} = \dfrac{x + 3}{4} \;\Rightarrow\; 4(x+1) = 2(x+3) \;\Rightarrow\; x = 1\]

Adding fractions — use LCM: \[\dfrac{x}{2} + \dfrac{x}{3} = 5 \;\xrightarrow{\times 6}\; 3x + 2x = 30 \;\Rightarrow\; x = 6\]

Forming Equations from Words

Step-by-step:

  1. Define your variable — let \(x = \ldots\)
  2. Translate each condition into algebra
  3. Form the equation
  4. Solve and check your answer makes sense

Example: Three consecutive integers sum to 51. \[x + (x+1) + (x+2) = 51 \;\Rightarrow\; 3x + 3 = 51 \;\Rightarrow\; x = 16\] Numbers: 16, 17, 18. ✓
