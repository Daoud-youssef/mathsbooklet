4.1 🎯 Learning Targets

By the end of this unit, you should be able to:

LT1 — Averages and Range from a List

  • Calculate the mean, median, mode and range from a list of values
  • Find a missing value given the mean of a dataset
  • Find the new mean when a value is added to a dataset
  • Find two unknown values given their ratio and the overall mean
  • Read and interpret a stem and leaf diagram to find averages and range

LT2 — Frequency Tables (Ungrouped)

  • Calculate the mean from a frequency table using \(\bar{x} = \dfrac{\sum fx}{\sum f}\)
  • Find the median from a frequency table using cumulative frequencies
  • Identify the mode from a frequency table
  • Find the largest and smallest possible value of an unknown frequency \(x\) given a target median

LT3 — Grouped Frequency Tables

  • Identify the modal class from a grouped frequency table
  • Identify the median class from a grouped frequency table
  • Estimate the mean using midpoints: \(\bar{x} \approx \dfrac{\sum fx}{\sum f}\)
  • Calculate a combined mean when two groups with different sizes are merged
  • Find missing frequencies \(a\) and \(b\) given total frequency and estimated mean

LT4 — Cumulative Frequency

  • Build a cumulative frequency table from a frequency table
  • Read values from a cumulative frequency table (how many below / above a value)
  • Use a cumulative frequency graph to estimate \(Q_1\), \(Q_2\) (median) and \(Q_3\)
  • Estimate the IQR from a cumulative frequency graph
  • Estimate the number of items above a given value
  • Find missing values A and B in a cumulative frequency table

LT5 — Box Plots and IQR

  • Calculate the IQR from raw data: \(\text{IQR} = Q_3 - Q_1\)
  • Draw a box-and-whisker plot from a five-number summary
  • Identify outliers using the rule: value \(> Q_3 + 1.5 \times \text{IQR}\) or \(< Q_1 - 1.5 \times \text{IQR}\)
  • Find unknown values in an ordered list given conditions on median, mode and range

4.2 🗂️ Formula Cards

Statistics — Key Formulas & Rules

Averages and Range

💡 Pro Tip Always sort your data first before finding the median or quartiles.

Mean: \[\bar{x} = \frac{\sum x}{n} = \frac{\text{sum of all values}}{\text{number of values}}\]

Median: Middle value when data is ordered.
If \(n\) is even: mean of the two middle values.

Mode: Most frequently occurring value.

Range: \[\text{Range} = \text{maximum} - \text{minimum}\]

Example: Data: \(3, 7, 7, 9, 14\) \[\bar{x} = \frac{3+7+7+9+14}{5} = 8 \qquad \text{Median} = 7 \qquad \text{Mode} = 7 \qquad \text{Range} = 11\]

Mean from a Frequency Table

🎯 Memory Trick Multiply each value by its frequency first — then divide by the total frequency, not the number of rows!

\[\bar{x} = \frac{\sum fx}{\sum f}\]

\(x\) \(f\) \(fx\)
2 3 6
4 7 28
6 5 30
Total 15 64

\[\bar{x} = \frac{64}{15} \approx 4.3\]

Grouped Frequency Tables

⚠️ Common Mistake Use the midpoint of each class — not the lower or upper boundary!

Estimated mean: \[\bar{x} \approx \frac{\sum fx}{\sum f} \quad \text{where } x = \text{midpoint of class}\]

Modal class: Class with the highest frequency.

Median class: Class containing the \(\dfrac{n}{2}\)th value (use cumulative frequency).

Example: Class \(10 \leq x < 20\), frequency \(= 8\) → midpoint \(x = 15\), so \(fx = 120\)

Cumulative Frequency

📝 Quick Method Read across from the CF axis to the curve, then down to the \(x\)-axis.

Building CF table: Each row = previous cumulative total \(+\) current frequency.

Reading quartiles from a CF graph (total = \(n\)):

Measure Read at CF value
Lower quartile \(Q_1\) \(\dfrac{n}{4}\)
Median \(Q_2\) \(\dfrac{n}{2}\)
Upper quartile \(Q_3\) \(\dfrac{3n}{4}\)

\[\text{IQR} = Q_3 - Q_1\]

Box Plots and Outliers

Five-number summary: Min, \(Q_1\), Median, \(Q_3\), Max

Box plot: Box spans \(Q_1\) to \(Q_3\). Median line inside. Whiskers extend to min and max.

Outlier rule: \[\text{Outlier if: } x > Q_3 + 1.5 \times \text{IQR} \quad \text{or} \quad x < Q_1 - 1.5 \times \text{IQR}\]

Example: \(Q_1 = 35,\ Q_3 = 55,\ \text{IQR} = 20\)
Upper fence \(= 55 + 1.5 \times 20 = 85\)
Any value \(> 85\) is an outlier.

Combined Mean

💡 Pro Tip You cannot just average two means — you must use the totals!

\[\bar{x}_{\text{combined}} = \frac{n_1 \bar{x}_1 + n_2 \bar{x}_2}{n_1 + n_2}\]

Example: Group 1: \(n=100\), mean \(= 18.2\). Group 2: \(n=50\), mean \(= 17.3\) \[\bar{x} = \frac{100 \times 18.2 + 50 \times 17.3}{150} = \frac{1820 + 865}{150} = \frac{2685}{150} = 17.9\]
