3.2 🗂️ Formula Cards

Unit 3: Mensuration — Formula Reference

📏 Unit Conversions

💡 Key Rule When converting units:
• Length → multiply or divide by the factor
• Area → use the factor squared
• Volume → use the factor cubed

Length chain: \(\text{km} \xrightarrow{\times 1000} \text{m} \xrightarrow{\times 100} \text{cm} \xrightarrow{\times 10} \text{mm}\)

Area chain: \(\text{km}^2 \xrightarrow{\times 10^6} \text{m}^2 \xrightarrow{\times 10^4} \text{cm}^2 \xrightarrow{\times 100} \text{mm}^2\)

Volume chain: \(\text{km}^3 \xrightarrow{\times 10^9} \text{m}^3 \xrightarrow{\times 10^6} \text{cm}^3 \xrightarrow{\times 10^3} \text{mm}^3\)

Capacity: \(1\text{ m}^3 = 1000\text{ L}\) \(\qquad\) \(1\text{ cm}^3 = 1\text{ mL}\)

⚠️ Common Error \(1\text{ m}^2 = 100\text{ cm}^2\) ✗ — should be 10 000 cm².
\(1\text{ m}^3 = 100\text{ cm}^3\) ✗ — should be 1 000 000 cm³.

📦 Volume Formulae

🎯 Core Idea For ANY prism or cylinder:
\[V = A_{\text{cross-section}} \times \text{length}\]

Solid Cross-section Formula
Rectangular prism Rectangle \(V = l \times w \times h\)
Triangular prism Triangle \(V = \tfrac{1}{2}\,b\,h\,l\)
Trapezoidal prism Trapezoid \(V = \tfrac{1}{2}(a+b)\,h\,l\)
Cylinder Circle \(V = \pi r^2 h\)
Half-cylinder Semicircle \(V = \tfrac{1}{2}\pi r^2 h\)
Quarter-cylinder Quarter circle \(V = \tfrac{1}{4}\pi r^2 h\)
Cylindrical sector Sector (\(\theta°\)) \(V = \dfrac{\theta}{360}\,\pi r^2 h\)

Composite solids: \(V_{\text{total}} = V_1 + V_2\) (add) or \(V_1 - V_2\) (subtract for holes)

⚠️ Common Errors
- Using diameter instead of radius: \(r = d \div 2\) first!
- Missing the \(\tfrac{1}{2}\) for triangular cross-sections.
- Adding volumes when you should subtract (hollow solids).

🎨 Surface Area Formulae

📝 Strategy Surface Area = sum of areas of ALL outer faces.
Sketch the solid, label every face, calculate each, then add.

Cuboid (all 6 faces): \[SA = 2(lw + lh + wh)\]

Open-top box (5 faces — no lid): \[SA = lw + 2lh + 2wh\]

Triangular prism (5 faces): \[SA = 2 \times A_{\triangle} + (b + s_1 + s_2) \times l\]

Trapezoidal prism (6 faces): \[SA = 2 \times \tfrac{1}{2}(a+b)h + (a + b + s_1 + s_2) \times l\]

Closed cylinder (curved + 2 circles): \[SA = 2\pi rh + 2\pi r^2\]

Open-top cylinder (curved + 1 circle): \[SA = 2\pi rh + \pi r^2\]

Tube/pipe (curved only, both ends open): \[SA = 2\pi rh\]

Half-cylinder (3 regions): \[SA = \underbrace{\pi r h}_{\text{half-curved}} + \underbrace{2rh}_{\text{flat rect}} + \underbrace{\pi r^2}_{\text{2 semicircles}}\]

💡 Cylinder Unrolled The curved surface of a cylinder unrolls into a rectangle:
width \(= 2\pi r\) (the circumference), height \(= h\).
So curved \(SA = 2\pi r \times h = 2\pi rh\).

⚠️ Common Errors — Surface Area
- Using the full cylinder SA formula when the cylinder is open at one or both ends.
- Forgetting to include both triangular ends of a triangular prism.
- Counting an internal join face that is not on the outside surface.
- Confusing SA (cm²) with Volume (cm³).

🔺 Pyramids, Cones and Spheres

🎯 Key Pattern Pyramids and cones are exactly ⅓ the volume of the prism or cylinder with the same base and height.

Solid Volume Surface Area
Pyramid (any base) \(V = \dfrac{1}{3} A_{\text{base}} \times h\) sum of all faces
Rectangular pyramid \(V = \dfrac{1}{3} lwh\)
Square pyramid \(V = \dfrac{1}{3} s^2 h\)
Cone \(V = \dfrac{1}{3}\pi r^2 h\) \(SA = \pi r^2 + \pi r l\)
Sphere \(V = \dfrac{4}{3}\pi r^3\) \(SA = 4\pi r^2\)
Hemisphere \(V = \dfrac{2}{3}\pi r^3\) \(SA = 3\pi r^2\)

Cone slant height: \(l = \sqrt{r^2 + h^2}\) (Pythagoras — slant is the hypotenuse)
Remember: \(r = d \div 2\) always before substituting.

⚠️ Common Errors — Curved Solids
- Forgetting the \(\tfrac{1}{3}\) in pyramid and cone formulas.
- Squaring \(r\) when you should cube it (sphere formula uses \(r^3\)).
- Using \(\tfrac{4}{3}\) instead of \(\tfrac{2}{3}\) for hemisphere volume.
- Using the perpendicular height \(h\) instead of slant \(l\) in cone SA.

🔣 Algebraic Volume

📝 Method Substitute the algebraic expressions into the volume formula, then expand brackets and collect like terms using index laws.

Steps:

  1. Write the formula: e.g. \(V = \tfrac{1}{2} \times \text{base} \times h \times l\)
  2. Substitute each expression: e.g. base \(= 2x\), \(h = x\), \(l = (x+4)\)
  3. Multiply step by step: \(V = \tfrac{1}{2} \times 2x \times x \times (x+4) = x^2(x+4)\)
  4. Expand: \(V = x^3 + 4x^2\)

Index law reminder: \((ax)^2 = a^2x^2\) — square both the coefficient AND the variable.

Example — solve for x: If \(V = 3x^3 = 375\), then \(x^3 = 125\), so \(x = \sqrt[3]{125} = 5\).

⚠️ Common Errors — Algebraic Volume
- \((2x)^2 = 4x^2\) ✓ not \(2x^2\) ✗
- Stopping before expanding all brackets.
- Forgetting \(V =\) on the left-hand side of the formula.
- Square rooting when the equation requires a cube root.
